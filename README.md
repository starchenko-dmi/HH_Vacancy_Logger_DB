# Парсер вакансий с hh.ru

Приложение для сбора данных о компаниях и их вакансиях с сайта [hh.ru](https://hh.ru) через официальное API, сохранения информации в локальной PostgreSQL-базе и выполнения базовых аналитических запросов через консольный интерфейс.

## Возможности

- Получение информации о работодателях по их ID на hh.ru.
- Сбор всех публичных вакансий указанных компаний (до 2000 на компанию).
- Сохранение данных в PostgreSQL: компании и вакансии.
- Интерактивное меню для:
  - Просмотра списка компаний и количества их вакансий.
  - Отображения всех вакансий с указанием зарплаты и ссылки.
  - Расчёта средней зарплаты по всем вакансиям (где указана зарплата).
  - Вывода вакансий с зарплатой выше средней.
  - Поиска вакансий по ключевому слову в названии.

## Требования

- **Python** 3.13
- **PostgreSQL** 12 или выше (должен быть запущен локально или доступен по сети)
- Установленные Python-пакеты:
  - `requests`
  - `psycopg2-binary`
  - `python-dotenv`
